# Base de Dados em SQL

Projeto de base de dados desenvolvido em SQL utilizando MySQL.

## Objetivo

Este projeto simula um sistema escolar contendo:

- Alunos
- Cursos
- Instrutores
- Matrículas
- Notas

## Tecnologias Utilizadas

- SQL
- MySQL
- Visual Studio Code
- Git
- GitHub

## Funcionalidades

- Criação de tabelas
- Relacionamentos com FOREIGN KEY
- INSERT, UPDATE e DELETE
- JOINs
- Transactions
